# weakpolviz

Interactive visualization of the degeneration graphs $R(\underline{h})$, $R_k(\underline{h})$, $R_k^{\circ}(\underline{h})$ underlying **polarized** and **weak polarized** relations on
admissible Hodge–Deligne diamonds with Hodge vector $\underline{h}=(h^0,h^1,\dots,h^1,h^0)$ of arbitrary weight $r$.[^1] 

## Run

Any static file server pointed at this folder, e.g.

```
python3 -m http.server 8100     # then open http://localhost:8100/
```
[^1]: Sourced from my paper with Matt Kerr [*Hodge Adjacency Conditions for Singularities*](https://arxiv.org/pdf/2505.09122), [PAMQ; February 17, 2026](https://intlpress.com/JFirstDetail/2013683611895697409).
