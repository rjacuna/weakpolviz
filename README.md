# weakpolviz

Interactive visualization of the degeneration graphs $R(h)$, $R_k(h)$, $R_k^∘(h)$ underlying **polarized** and **weak polarized** relations on
admissible Hodge–Deligne diamonds from my paper with Matt Kerr [*Hodge Adjacency Conditions for Singularities*](https://arxiv.org/pdf/2505.09122), [PAMQ; February 17, 2026](https://intlpress.com/JFirstDetail/2013683611895697409)).

## Run

Any static file server pointed at this folder, e.g.

```
python3 -m http.server 8100     # then open http://localhost:8100/
```
